======================================================
PROJECT: Interactive Quiz Game (เกมตอบคำถามประลองเชาว์)
======================================================

1. รายละเอียดโปรเจกต์ (Project Description)
   เกมตอบคำถามบนเว็บ (Web-based Quiz Game) ที่พัฒนาโดยใช้ HTML, CSS และ JavaScript 
   เน้นประสบการณ์ผู้ใช้งาน (User Experience) ที่ลื่นไหล มีการตอบสนองทันทีเมื่อกดตอบ 
   และมีการใช้ Library ภายนอกเพื่อความสวยงาม

2. หัวข้อที่เลือก (Selected Topic)
   "ปัญหาเชาว์และคำถามกวนโอ๊ย" (Funny Riddles & Trick Questions)
   - เป็นชุดคำถาม 10 ข้อที่เน้นความตลกและต้องใช้ไหวพริบในการสังเกต
   - มีการเฉลยคำตอบพร้อมมุขตลกในตัวเลือก

3. ฟีเจอร์พิเศษที่เพิ่มเข้ามา (Special Features)
   
   [Functionality & Logic]
   ★ ระบบจับเวลา (Countdown Timer): 
      - มีการนับถอยหลังตลอดการเล่นเกม เพื่อเพิ่มความท้าทาย
      - หากเวลาหมด เกมจะตัดจบและสรุปคะแนนทันที
      - ใช้ JavaScript Date() object เพื่อความแม่นยำของเวลา
   
   ★ ระบบเฉลยอัตโนมัติ (Instant Feedback):
      - เมื่อตอบถูก: ปุ่มเปลี่ยนเป็นสีเขียว พร้อม Animation แสดงความยินดี
      - เมื่อตอบผิด: ปุ่มเปลี่ยนเป็นสีแดง พร้อม "สั่น" เตือน และระบบจะ "เฉลย" ข้อที่ถูกด้วยสีเหลืองให้ผู้เล่นรู้ทันที

   [Design & User Experience]
   ★ การใช้ Animation (Animate.css):
      - เพิ่มลูกเล่นให้ปุ่มตอบคำถาม (Effect: Tada / Shake)
      - Animation การเปิดตัวของ Toast แจ้งเตือนและ Modal สรุปผล
   
   ★ Bootstrap 5 Integration:
      - ใช้ Bootstrap Components เต็มรูปแบบ (Modal, Toast, Buttons, Layout)
      - หน้าเว็บรองรับการแสดงผลบนมือถือ (Responsive Design)

   ★ ระบบสรุปผล (Scoring System):
      - แสดงคะแนนรวมเมื่อจบเกม
      - มีข้อความ Feedback ให้กำลังใจผู้เล่นที่แตกต่างกันตามช่วงคะแนนที่ได้
